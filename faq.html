Frequently Asked Questions

Jawaban untuk pertanyaan yang sering diajukan tentang ArusdataJS.

Pertanyaan Umum

Apa itu ArusdataJS?

ArusdataJS adalah library JavaScript untuk komunikasi real-time yang ringan, cepat, dan mudah digunakan. Library ini memungkinkan developer untuk dengan mudah menambahkan fitur real-time seperti chat, notifikasi, dan pembaruan data langsung ke aplikasi web dan mobile.

Apakah ArusdataJS gratis untuk digunakan?

Ya, ArusdataJS adalah open source dan gratis untuk digunakan. Library ini dirilis di bawah lisensi MIT, yang memungkinkan penggunaan untuk proyek komersial dan non-komersial tanpa biaya.

Apa perbedaan ArusdataJS dengan Socket.IO atau Pusher?

ArusdataJS menawarkan solusi yang lebih ringan dan sederhana dibandingkan Socket.IO, dengan fokus pada kemudahan penggunaan dan performa tinggi. Dibandingkan dengan Pusher, ArusdataJS bisa di-host sendiri (self-hosted) dan tidak memerlukan biaya langganan. Namun, ArusdataJS memiliki fitur yang mirip seperti channels, event-based messaging, dan authentication.

Pertanyaan Teknis

Bagaimana ArusdataJS menangani koneksi yang terputus?

ArusdataJS memiliki mekanisme reconnect otomatis yang akan mencoba menghubungkan kembali jika koneksi terputus. Anda bisa mengatur interval reconnect melalui opsi konfigurasi: 

const arusdata = new ArusdataClient({
  endpoint: 'ws://localhost:8080/arusdata',
  reconnectInterval: 5000, // Coba reconnect setiap 5 detik
  autoReconnect: true      // Aktifkan reconnect otomatis
});

Apakah ArusdataJS mendukung SSL/TLS?

Ya, ArusdataJS mendukung koneksi WebSocket aman melalui WSS (WebSocket Secure). Untuk menggunakan koneksi aman, cukup gunakan protokol "wss://" di URL endpoint saat inisialisasi client: 

const arusdata = new ArusdataClient({
  endpoint: 'wss://your-server.com/arusdata'
});

Berapa banyak koneksi yang bisa ditangani server ArusdataJS?

Jumlah koneksi yang bisa ditangani tergantung pada kapasitas server dan konfigurasi sistem operasi. Pada umumnya, sebuah server dengan spesifikasi standar dapat menangani ribuan koneksi secara bersamaan. Untuk skala yang lebih besar, ArusdataJS bisa dikonfigurasi dengan load balancer dan multiple server instances.

Bagaimana cara mengotentikasi pengguna untuk private channels?

Private channels menggunakan sistem otentikasi berbasis token. Server membuat token berdasarkan socket ID dan nama channel, dan client harus memberikan token ini saat berlangganan ke private channel: 

// Client-side
async function subscribeToPrivateChannel(channelName) {
  // Dapatkan token autentikasi dari server
  const response = await fetch('/auth', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ 
      channel_name: channelName, 
      socket_id: arusdata.getSocketId() 
    })
  });
  
  const data = await response.json();
  
  // Subscribe ke private channel
  const channel = arusdata.subscribe(channelName);
  channel.auth = data.auth;
  
  return channel;
}

Apakah ArusdataJS mendukung horizontal scaling?

Ya, ArusdataJS bisa dikonfigurasikan untuk bekerja dengan arsitektur terdistribusi. Untuk mengimplementasikan horizontal scaling, Anda perlu menggunakan adapter penyimpanan eksternal seperti Redis untuk mengelola shared state antara server instances. Dokumentasi lebih detail tentang konfigurasi ini tersedia di GitHub repository.

Penggunaan

Apakah ArusdataJS bekerja di semua browser?

ArusdataJS menggunakan WebSocket API yang didukung oleh semua browser modern. Secara khusus, ArusdataJS mendukung:

  • Chrome 16+
  • Firefox 11+
  • Safari 7+
  • Edge 12+
  • Opera 12.1+
  • iOS Safari 6.0+
  • Android Browser 4.4+

Apakah ArusdataJS bisa digunakan dengan React/Vue/Angular?

Tentu, ArusdataJS bisa digunakan dengan framework JavaScript apapun. Library ini tidak memiliki ketergantungan khusus dan bisa diintegrasikan dengan mudah ke project React, Vue, Angular, atau framework lainnya.

Bagaimana cara menggunakan ArusdataJS dengan Node.js di backend?

Server ArusdataJS berjalan di atas Node.js. Anda bisa menginstall server menggunakan npm: 

npm install arusdata-server --save

Kemudian jalankan server di aplikasi Node.js Anda: 

const ArusdataServer = require('arusdata-server');

const server = new ArusdataServer({
  port: 8080,
  path: '/arusdata'
});

// Gunakan dengan Express yang sudah ada
const express = require('express');
const http = require('http');
const app = express();
const httpServer = http.createServer(app);

const arusdata = new ArusdataServer({
  server: httpServer,
  path: '/arusdata'
});
