Client API

Constructor

new ArusdataClient(options)

Membuat instance baru ArusdataClient dengan opsi konfigurasi.

Parameters

Name Type Default Description
options.endpoint String 'ws://localhost:8080/arusdata' URL WebSocket server
options.reconnectInterval Number 3000 Interval percobaan reconnect dalam milidetik
options.autoReconnect Boolean true Reconnect otomatis saat koneksi terputus
options.autoConnect Boolean true Connect otomatis saat inisialisasi

Example

const arusdata = new ArusdataClient({
    endpoint: 'ws://myserver.com/arusdata',
    reconnectInterval: 5000,
    autoReconnect: true,
    autoConnect: true
});

Methods

connect()

Mencoba untuk terhubung ke server WebSocket.

Returns

void

Example

// Memastikan client terhubung
arusdata.connect();
disconnect()

Memutuskan koneksi dari server WebSocket.

Returns

void

Example

// Memutuskan koneksi
arusdata.disconnect();
subscribe(channelName)

Subscribe ke channel.

Parameters

Name Type Description
channelName String Nama channel yang ingin disubscribe

Returns

Channel - Object channel dengan method-method untuk berinteraksi dengan channel.

Example

// Subscribe ke channel
const channel = arusdata.subscribe('chat-room');
unsubscribe(channelName)

Unsubscribe dari channel.

Parameters

Name Type Type Description
channelName String Nama channel yang ingin di-unsubscribe

Returns

void

Example

// Unsubscribe dari channel
arusdata.unsubscribe('chat-room');
publish(channelName, eventName, data)

Mempublikasikan event ke channel.

Parameters

Name Type Description
channelName String Nama channel tujuan
eventName String Nama event yang dipublikasikan
data Any Data yang dikirim dengan event

Returns

Boolean - true jika berhasil, false jika gagal

Example

// Publish event ke channel
arusdata.publish('chat-room', 'new-message', {
    sender: 'John',
    text: 'Hello world!'
});
onConnection(callback)

Mendaftarkan callback untuk event koneksi berhasil.

Parameters

Name Type Description
callback Function Fungsi yang dipanggil saat koneksi berhasil, menerima socketId sebagai parameter

Returns

ArusdataClient - Instance client untuk method chaining

Example

// Mendaftarkan callback koneksi
arusdata.onConnection((socketId) => {
    console.log('Connected with ID:', socketId);
});
onDisconnection(callback)

Mendaftarkan callback untuk event koneksi terputus.

Parameters

Name Type Description
callback Function Fungsi yang dipanggil saat koneksi terputus

Returns

ArusdataClient - Instance client untuk method chaining

Example

// Mendaftarkan callback disconnection
arusdata.onDisconnection(() => {
    console.log('Disconnected from server');
});
getSocketId()

Mendapatkan ID socket dari koneksi yang sedang berlangsung.

Returns

String|null - ID socket jika terhubung, null jika belum terhubung

Example

// Mendapatkan socket ID
const socketId = arusdata.getSocketId();
isConnected()

Memeriksa apakah client terhubung ke server.

Returns

Boolean - true jika terhubung, false jika tidak

Example

// Memeriksa status koneksi
if (arusdata.isConnected()) {
    console.log('Connected to server');
} else {
    console.log('Not connected');
}

Events

Client events dapat didengarkan menggunakan method onConnection dan onDisconnection. Berikut adalah daftar events yang tersedia:

Event Description Data
connection Terjadi saat koneksi ke server berhasil dibuat socketId (String) - ID socket dari koneksi
disconnection Terjadi saat koneksi ke server terputus null

Channel API

Methods

bind(eventName, callback)

Mendaftarkan callback untuk dipanggil saat event tertentu terjadi pada channel.

Parameters

Name Type Description
eventName String Nama event yang ingin didengarkan
callback Function Fungsi yang dipanggil saat event terjadi, menerima data event sebagai parameter

Returns

Channel - Instance channel untuk method chaining

Example

// Binding event
channel.bind('message', (data) => {
console.log('Received message:', data);
});
unbind(eventName, callback)

Menghapus callback dari event tertentu.

Parameters

Name Type Description
eventName String Nama event yang ingin dihapus callbacknya
callback Function (optional) Callback spesifik yang ingin dihapus. Jika tidak ditentukan, semua callback untuk event tersebut akan dihapus.

Returns

Channel - Instance channel untuk method chaining

Example

// Simpan referensi callback
const messageCallback = (data) => {
console.log('Received message:', data);
};

// Bind callback
channel.bind('message', messageCallback);

// Unbind callback tertentu
channel.unbind('message', messageCallback);

// Atau unbind semua callback untuk event tertentu
channel.unbind('message');
trigger(eventName, data)

Mempublikasikan event ke channel.

Parameters

Name Type Description
eventName String Nama event yang ingin dipublikasikan
data Any Data yang ingin dikirim dengan event

Returns

Boolean - true jika berhasil, false jika gagal

Example

// Mengirim event ke channel
channel.trigger('message', {
text: 'Hello world!',
timestamp: new Date().toISOString()
});
onSubscription(callback)

Mendaftarkan callback yang dipanggil saat subscription berhasil.

Parameters

Name Type Description
callback Function Fungsi yang dipanggil saat subscription berhasil, menerima data subscription sebagai parameter

Returns

Channel - Instance channel untuk method chaining

Example

// Mendaftarkan callback subscription
channel.onSubscription((data) => {
console.log('Successfully subscribed with data:', data);
});

Events

Channel secara otomatis menangani beberapa event sistem. Selain itu, Anda dapat menangani event kustom yang dikirim dari server atau client lain.

System Events

Event Description Data
subscription_succeeded Terjadi saat subscription ke channel berhasil { subscribers: Number }
subscription_error Terjadi jika ada error saat mencoba berlangganan ke channel { message: String }

Custom Events

Selain system events, Anda dapat menerima dan mengirim event kustom dengan nama apapun. Event kustom dibind menggunakan method bind() dan dipanggil dengan method trigger().

Server API

Constructor

new ArusdataServer(options)

Membuat instance baru ArusdataServer dengan opsi konfigurasi.

Parameters

Name Type Default Description
options.port Number 8080 Port HTTP server
options.path String '/arusdata' Path WebSocket endpoint
options.pingInterval Number 30000 Interval ping/pong dalam milidetik
options.server HTTPServer null HTTP server yang sudah ada (opsional)
options.authSecret String 'arusdata-secret-key' Secret key untuk otentikasi private channels

Example

const ArusdataServer = require('arusdata-server');

// Basic configuration
const server = new ArusdataServer({
port: 8080,
path: '/arusdata',
authSecret: 'my-secret-key'
});

// Advanced configuration with existing Express app
const express = require('express');
const http = require('http');

const app = express();
const httpServer = http.createServer(app);

const arusdataServer = new ArusdataServer({
server: httpServer,
path: '/realtime',
pingInterval: 60000
});

httpServer.listen(3000);

Methods

broadcastToChannel(channelName, eventName, data, excludeClient)

Mengirim event ke semua client yang subscribe ke channel tertentu.

Parameters

Name Type Default Description
channelName String - Nama channel tujuan
eventName String - Nama event yang dikirim
data Any - Data yang dikirim dengan event
excludeClient WebSocket null Client yang dikecualikan dari broadcast (opsional)

Returns

Number - Jumlah client yang menerima pesan

Example

// Broadcast ke semua client dalam channel
const recipientCount = channelsController.broadcastToChannel(
'chat-room',
'new-message',
{ sender: 'System', text: 'Hello everyone!' }
);

console.log(`Message sent to ${recipientCount} recipients`);
getStats()

Mendapatkan statistik server.

Returns

Object - Objek yang berisi statistik server

Example

// Mendapatkan statistik server
const stats = server.getStats();

console.log(`Active connections: ${stats.activeConnections}`);
console.log(`Total messages processed: ${stats.messagesProcessed}`);

Events

Server menghasilkan beberapa event internal. Anda dapat mendaftarkan event listeners untuk merespons event ini.

Event Description Data
client_connected Terjadi saat client terhubung ke server { client: WebSocket, ip: String, clientId: String }
client_disconnected Terjadi saat client terputus dari server { clientId: String }
channel_subscribe Terjadi saat client berlangganan ke channel { client: WebSocket, channel: String }
channel_unsubscribe Terjadi saat client berhenti berlangganan dari channel { client: WebSocket, channel: String }
message_published Terjadi saat pesan dipublikasikan ke channel { client: WebSocket, channel: String, event: String, data: Any }

Authentication

ArusdataJS menggunakan sistem otentikasi berbasis token untuk private channels. Berikut adalah alur proses otentikasi:

Authentication Flow

  1. Client terhubung ke server dan menerima socket_id unik
  2. Ketika client ingin berlangganan private channel, client meminta token otentikasi dari server melalui HTTP endpoint /auth
  3. Server memvalidasi apakah client memiliki izin untuk mengakses channel tersebut
  4. Jika diizinkan, server mengembalikan token otentikasi yang dihitung dengan rumus: HMAC(secret_key, socket_id + ':' + channel_name)
  5. Client berlangganan ke private channel dengan menyertakan token otentikasi
  6. Server memverifikasi token dan mengizinkan subscription jika valid

Authentication Request Format

POST /auth HTTP/1.1
Content-Type: application/json

{
"channel_name": "private-messages",
"socket_id": "client-12345"
}

Authentication Response Format

HTTP/1.1 200 OK
Content-Type: application/json

{
"auth": "a8b25dc1d939b0d16063b500b2bbert15e55b2ert2c5f7dba1e3b9c3afc6"
}

Authentication Code Example

// Server-side (Node.js) authentication endpoint
app.post('/auth', express.json(), (req, res) => {
const { channel_name, socket_id } = req.body;

if (!channel_name || !socket_id) {
return res.status(400).json({ error: 'Missing required parameters' });
}

// Validate the user's access to this channel (example)
if (!userHasAccessToChannel(req.user, channel_name)) {
return res.status(403).json({ error: 'Access denied' });
}

// Generate authentication signature
const stringToSign = `${socket_id}:${channel_name}`;
const signature = crypto
.createHmac('sha256', config.authSecret)
.update(stringToSign)
.digest('hex');

// Return the auth token
res.status(200).json({
auth: signature
});
});
