Komunikasi Real-time Sederhana dan Handal

ArusdataJS adalah library JavaScript untuk membangun aplikasi real-time yang ringan, cepat, dan mudah digunakan. Tanpa dependency eksternal.

Mulai Sekarang Lihat Demo
arusdata-example.js
// Connect to ArusdataJS server
const arusdata = new ArusdataClient({
    endpoint: 'wss://arusdata-server.up.railway.app/arusdata'
});

// Subscribe to a channel
const channel = arusdata.subscribe('my-channel');

// Listen for events
channel.bind('new-message', (data) => {
    console.log(`New message: ${data.text}`);
    displayMessage(data);
});

// Send a message
channel.trigger('new-message', {
    sender: 'John',
    text: 'Hello world!'
});

Fitur Utama

Ringan & Cepat

Ukuran library yang kecil (~10KB gzipped) dengan kinerja tinggi untuk aplikasi real-time.

Mudah Diintegrasikan

API yang intuitif dan dokumentasi lengkap membuat integrasi menjadi sangat mudah.

Tanpa Dependency

Tidak memerlukan library eksternal, mengurangi ukuran bundle aplikasi Anda.

Aman

Dukungan otentikasi dan otorisasi untuk private channels.

Server Terintegrasi

Server WebSocket Node.js yang siap pakai dan mudah dikonfigurasi.

Open Source

Kode sumber terbuka dan tersedia di GitHub untuk dimodifikasi sesuai kebutuhan.

Quick Start Guide

// Inisialisasi client
const arusdata = new ArusdataClient({
  endpoint: 'wss://arusdata-server.up.railway.app/arusdata'
});

// Subscribe ke channel
const channel = arusdata.subscribe('chat');

// Bind ke event
channel.bind('message', (data) => {
  console.log('Received message:', data);
});

// Publish event
channel.trigger('message', {
  sender: 'Username',
  text: 'Hello, ArusdataJS!'
});

// Callback saat koneksi berhasil
arusdata.onConnection((socketId) => {
  console.log('Connected with socket ID:', socketId);
});

// Callback saat koneksi terputus
arusdata.onDisconnection(() => {
  console.log('Disconnected from server');
});
