Release Notes

Catatan perubahan dan pembaruan ArusdataJS untuk setiap versi.

Penjelasan Versioning

ArusdataJS mengikuti aturan Semantic Versioning (SemVer):

  • Major version (x.0.0) - Perubahan yang tidak kompatibel dengan versi sebelumnya
  • Minor version (0.x.0) - Penambahan fitur yang kompatibel dengan versi sebelumnya
  • Patch version (0.0.x) - Bug fixes dan perbaikan kecil yang kompatibel dengan versi sebelumnya

Version 0.1.0 (Current)

Latest

Released on March 26, 2025

Initial public release of ArusdataJS, providing core functionality for real-time messaging.

Features

  • WebSocket server for real-time communication
  • Client-side JavaScript library for browser integration
  • Channel-based communication system
  • Event-driven messaging model
  • Automatic reconnection with configurable intervals
  • Private channels with authentication
  • Presence channels for user status tracking
  • REST API for monitoring and external integrations

Documentation

  • Getting Started guide
  • API Reference
  • Example implementations
  • Interactive demos
